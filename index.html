HTML Notes

  • body - wraps the entire web page
  • h1 (up to h6) - header options
  • p - paragraph within the body
  • div - paragraph seperated by a line of space
  • strong - bolds text
  • em - italicises text
  • br - starting tag only line break
  • ul - unordered bullet point list
  • li - list item
  • ol - ordered list with numbers
  • img src="imagelocation.jpg" alt="description of image" / >
  • video src="" width"320" height="240" controls > Video not supported
  • head - contains metadata, goes under 'html'
  • title - sets title of a web page, goes under 'head'
  • < a href="https://www.site.com/" target="_blank" > This is a link to site < / a > - adds hyperlink to website
  • < a href="./contact.html" > Contact - new website page
  • < li > < a href="headername" >Header Reference< /a > < /li > - links to part of website
  • < ! -- -- > - add comment, not shown on website
